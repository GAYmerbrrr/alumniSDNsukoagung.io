LATAR BELAKANG PEMBUATAN WEBSITE INI

Di suatu hari, 4 hari menjelang ujian sekolah saya sedang berbaring di kursi, tapi alih-alih bersiap untuk menghadapi uijan sekolah, saya justru terpikirkan sebuah ide untuk membuat suatu website untuk mengenang masa SD kita, saya pun langsung mulai membuat website ini dengan HTML,CSS dan sedikit JavaScript hanya dengan menggunakan ponsel yang saya miliki.

Saya pun membuat website ini sampai larut malam sambil menonton pertandingan Piala Asia Qatar u23 antara timnas Indonesia menghadapi Australia, setelah itu saya pun tidur dan melanjutkan membuat website ini di jumat siang dan selesai di sore harinya.
